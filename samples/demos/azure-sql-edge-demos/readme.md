# Azure SQL Edge Demos

**Notice: Azure SQL Edge was retired on September 30, 2025.**

For historical reference and migration guidance, please see the [Azure SQL Edge retirement announcement](https://azure.microsoft.com/updates/v2/azure-sql-edge-retirement).

## Archived Content

Previously, this directory contained demo projects for Azure SQL Edge. These demos have been removed as the service is no longer supported. For historical reference, you can find the archived documentation at:

- [Azure SQL Edge Documentation (Previous Versions)](https://learn.microsoft.com/en-us/previous-versions/azure/azure-sql-edge/)
- [Tutorial: Deploy Azure resources](https://learn.microsoft.com/en-us/previous-versions/azure/azure-sql-edge/tutorial-deploy-azure-resources)
- [Tutorial: Set up IoT Edge modules](https://learn.microsoft.com/en-us/previous-versions/azure/azure-sql-edge/tutorial-set-up-iot-edge-modules)

## Migration Options

Consider these alternatives for edge database scenarios:
- **Azure SQL Database** for cloud-based applications
- **SQL Server on Linux** for on-premises edge scenarios
- **Azure Database for PostgreSQL** for open-source alternatives
- **Azure IoT Edge** with other database solutions
